# SyncPlayersFoodLevel
一个 Spigot/Paper 娱乐类插件，同步所有玩家的饥饿值！

README> 中文 | [English](.github/README_EN.MD)

------------

### 下载
通过 Release 下载稳定版或通过 Actions 中的 Artifacts 下载最新构建版。

### 使用
直接放入 `plugins` 文件夹即可，没有配置文件，没有命令。
